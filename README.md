# Glyph
A console text editor with text decoration.
## Keybinds
CTRL + X: exit \
CTRL + F: From \
CTRL + W: Marker \
CTRL + E: Text Color \
CTRL + Z: Save \
CTRL + B: Bold \
CTRL + I: Italic \
CTRL + U: Underline \
ESC: Deselect From \
Shift + Arrow: Scrolling \
Arrow: Cursor
